Based on all sources above, these industries have the strongest signal for unmet software demand at the micro-operator level:

1. **Pest control** — $30B TAM, regulatory complexity, recurring revenue model, $300M exit validates market
2. **HVAC/Plumbing** — $150B combined, PE consolidation driving software demand, multiple YC bets (Broccoli, Cactus, Drillbit)
3. **Junk removal / hauling** — fast-growing, zero dedicated tools, operators still price via phone calls
4. **Self-storage (small operators)** — 50K+ facilities, enterprise tools too expensive, 2005-era software standard
5. **HOAs (self-managed)** — 370K HOAs, volunteer board operators, no modern affordable tool
6. **Fleet maintenance (5-30 vehicles)** — every trades company has this pain, no tool in the $29-59/month range
7. **Commercial cleaning (B2B janitorial)** — bid calculation + recurring schedule management after winning contracts
8. **Septic service** — peak boring, paper-based records, recurring pumping reminders alone = clear value
9. **Lawn care routing + billing** — weather rescheduling + automated billing = 600K+ operators
10. **Trade schools** — enrollment booming, existing software built for universities not vocational programs
