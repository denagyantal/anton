# AI-Powered Quoting & Estimating for Trade Contractors

**Score: 84/105** | **Verdict: BUILD** | **Evaluation Date: 2026-03-05**

## Concept
Mobile-first quoting app for trade contractors. Enter address → auto-pull property data → select trade-specific line items → generate professional PDF → text/email to customer → collect e-signature → follow-up automation. "Quote from the driveway in 5 minutes." AI-assisted pricing from job history.

## Target Customer
Small trade contractors — HVAC, plumbing, electrical, painting, roofing, concrete, fencing. Currently using Word docs, Excel templates, or handwritten estimates. Average quote takes 45 minutes with 22% close rate; professional quotes achieve 43% close rate.

## Key Differentiators
1. **Auto-pull property data** — address input → lot size, sq footage, building type
2. **Trade-specific template library** — HVAC units, plumbing fixtures, electrical panels
3. **30-second professional quote** — phone-first, on-site generation
4. **E-signature + payment** — close the deal on the spot
5. **AI pricing suggestions** — learn from job history and market data

## LTD Pricing
- $59 LTD: Core quoting with templates and PDF generation
- $99 LTD: + property data auto-pull, AI pricing, follow-up automation

## Next Steps

2. Could be absorbed as a feature into FSM tools
3. AI-generated estimates need validation with real contractors
