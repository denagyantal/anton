## Market Context
- 40,000+ independent insurance agencies in the US; the overwhelming majority are under 5 agents
- AMS360 (Vertafore): opaque pricing, explicitly "too expensive for small agencies"
- HawkSoft: $94/user/mo flat base — a 3-person shop pays $300+/mo
- Applied Epic: enterprise-grade, inaccessible for small agencies ($500+/mo+)
- Jenesis: budget option but less reviewed, limited integrations
- No modern, affordable AMS at under $100/mo flat for 1–5 agent tier

## Core Feature Set (MVP)
1. Client + policy database (client profiles, policy details, carrier, premium, expiration)
2. Automated renewal reminders (email + SMS, 90/60/30 days before expiration)
3. Commission split calculator (agent splits, carrier statements, commission ledger)
4. Carrier document vault (quote comparisons, binders, certificates)
5. Book-of-business report (expiring policies view, revenue forecast)

## Key Differentiators
1. Flat pricing — no per-user fees (a 3-person shop pays the same as 1 person)
2. LTD option removes the monthly cost objection keeping small agencies on Excel
3. Modern UI vs. AMS360/HawkSoft dated interfaces
4. Zero implementation cost — agents can be up and running in an afternoon
