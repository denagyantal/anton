## Market

- Belgium: All VAT-registered businesses mandatory from Jan 1, 2026 (live)
- Poland: All businesses from Apr 2026 (live)
- France: Large/mid-size from Sep 2026; all SMEs Sep 2027
- 10M+ EU SMBs affected; Billentis projects 28% annual market growth → €22–24B by 2028

## Target ICP

- Belgian/Polish/French sole traders and micro-SMBs (1–5 employees)
- Invoicing 5–20 customers per month via PDF/Word today
- No IT staff; no enterprise accounting system
- Accountants serving these businesses (as resellers/recommenders)

## Product Concept

**MVP (2–3 weeks, Belgium-first):**
1. User enters their VAT number + invoice details (or pastes CSV)
2. Tool converts to Peppol BIS Billing 3.0 XML automatically
3. Submits to Peppol network or provides downloadable file for their accounting portal
4. Sends to buyer's network endpoint

**V2 features:**
- Connector for QBO/Xero/FreshBooks (auto-translate outgoing invoices)
- Poland UBL/KSeF format support
- France Factur-X format support
- Webhook API for SaaS platforms to embed compliance

## Key Differentiators

- Zero spec knowledge required — enter invoice details in plain language, get compliant XML
- Belgium-first (live mandate, English-speaking business community, real urgency NOW)
- Country expansion roadmap = moat (Belgium → Poland → France → Italy)
- No per-invoice fees — flat annual or one-time LTD
