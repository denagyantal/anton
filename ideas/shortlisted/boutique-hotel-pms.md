## One-Line Pitch
One-screen hotel management for 5–30 room properties — drag-and-drop calendar, OTA sync, and automated guest messages without enterprise complexity.

## Problem

Boutique hotels (5–30 rooms), bed & breakfasts, and small vacation rental portfolios are forced into enterprise property management systems (Cloudbeds $300+/month, Mews $200+/month, Little Hotelier $150+/month) with enterprise-tier complexity. HN commenters in the "boring businesses" thread explicitly described existing hotel software as "garbage" and "entrenched."

**Current workarounds**: Airtable spreadsheets for reservations, manual OTA channel updates (leading to double-bookings), email chains for check-in instructions, Google Sheets for housekeeping schedules.

**The core pain**: A 12-room B&B doesn't need 40 modules and 200 settings. They need: a calendar that shows who's where, OTA sync so they don't double-book, and automatic check-in messages. That tool doesn't exist at a reasonable price.

## Market Evidence

## Key Differentiators
1. **One-screen simplicity** — everything the owner sees on a single drag-and-drop calendar
2. **OTA sync** — Booking.com + Airbnb + Expedia availability in real-time; no manual updates, no double-bookings
3. **Automated guest communications** — check-in instructions, checkout reminder, post-stay review request
4. **Housekeeping schedule** — auto-generates from check-out dates; staff see daily task list on mobile
5. **Flat pricing** — $99/month (not per-room); no enterprise tiers, no module add-ons

## LTD Pricing
- $99 LTD: Up to 15 rooms, 2 OTA connections, 1 year of guest messages
- $149 LTD: Unlimited rooms, 3 OTA connections, full guest journey automation

## Target Channels
- Facebook groups: "Boutique Hotel Owners", "Bed and Breakfast Owners", "Inn Keepers Association"
- PAII (Professional Association of Innkeepers International) membership
- State B&B associations (every US state has one)
- AppSumo as primary launch channel

## Next Steps
1. Build MVP: reservation calendar (drag-and-drop) + OTA sync (Booking.com + Airbnb first via channel manager API) + automated check-in/out messages via email/SMS
2. Validate: post in boutique hotel FB groups asking "what's your #1 software complaint?" — target operators who say "Cloudbeds is overkill" or "I use Airtable for reservations"
3. Launch $99 LTD on AppSumo; first hotel software deal = category-first positioning
4. V2: housekeeping scheduler + direct booking widget for property website

## Top Risks
1. **OTA API complexity** — real-time availability sync with Booking.com, Airbnb, Expedia requires XML/API partnerships that take weeks to set up and maintain
2. **Incumbent switching cost** — properties on Cloudbeds have historical booking data and guest records; migration friction is real
3. **Seasonal demand** — most boutique hotels don't evaluate/switch software mid-season; must target off-season
4. **International tax/compliance** — VAT, tourist tax, local regulations vary by country; start US-only
