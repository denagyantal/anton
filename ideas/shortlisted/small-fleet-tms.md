## One-Line Pitch
ELD + IFTA + dispatch + load management in one flat $59/mo tool for 1–10 truck owner-operators — the anti-Motive.

## Problem
Owner-operators and micro-fleets (1–10 trucks) are priced out of enterprise TMS platforms. Motive/Samsara charge $35–70/vehicle/mo for ELD alone, with separate subscriptions required for IFTA reporting, dispatch, load boards, and fuel card management. ELD compliance is federally mandated for any CDL truck traveling interstate. IFTA quarterly filings are also mandatory. There is no single affordable tool combining compliance + dispatch + load tracking below $100/mo total for a solo operator.

## Market Evidence
- 3.5M+ registered CDL truck drivers in the US; ~1M owner-operators
- Motive (formerly KeepTruckin) $35–70/vehicle/mo; Samsara $35–65/vehicle/mo
- ELD mandate (2017) = non-discretionary purchase for all interstate CMVs
- IFTA quarterly filing = non-discretionary for multi-state operators
- FleetRabbit ($3/vehicle/mo) validates demand for ultra-affordable fleet tools
- Trucking companies cited across r/Truckers as "most underserved by affordable software"

## Key Differentiators
1. Flat $59/mo for 1–10 trucks — no per-vehicle overage penalty
2. ELD + IFTA + dispatch in one tool (no 3-app cobbling)
3. Automated IFTA quarterly report generation (fuel tax filing)
4. Load board integration (DAT/Truckstop) + owner payment tracking
5. Mobile-first for solo operator managing their own truck

## Next Steps
1. Validate ELD hardware API options (existing ELD device integrations vs. proprietary)

## Risks
1. ELD hardware compliance (FMCSA-registered device required) adds certification overhead
2. Motive/Samsara have huge brand awareness and driver familiarity
3. Load board API costs (DAT ~$100+/mo) could eat margins
4. Regulatory changes in ELD specs require ongoing maintenance
